Manchester, UK

Software developer and hacker of code. I absolutely love making things. Mostly I make things for the lovely Melbourne Hosting in Manchester. I contribute to open source projects on GitHub when I can.

Day to day, I tend to write web apps in Ruby using Rails. I like to get my hands dirty with the whole web stack, HTML/CSS through JavaScript, into the application layer server-side, through the web server and the database layer and through to the system itself.

I enjoy clean, simple, loveable design.

I spent the first decade or so of my screen time using a variant of Windows. These days you will find me working on a variant of Unix.

Lately I spend a lot of my time organising events in the technical community. 2011 was @MagRails, this year is @HackManchester, next year is @BritRuby.

I believe clear, regular, focussed communication is more worthwhile to creating great software than any tool or process.

I like getting involved with new things. Talk to me, if you think I might be interested in something you're doing.
